# pasteMEAN

A Pastebin clone made with the MEAN stack (MongoDB, ExpressJS, AngularJS, NodeJS)


## Why?

I wanted to learn the MEAN stack and this was a nice way to start.


## How do I use this?

* Install MongoDB
* Install npm
* Install bower
* Clone/Fork this
* Run "npm install"
* Run "bower install"
* Enjoy
